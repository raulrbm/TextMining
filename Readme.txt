Readme

El archivo pt.py aplica el POS-tagger de nltk a los textos de la carpeta corp, generando los archivos contenidos en la carpeta poscorp.

El archivo buildDataset.py genera el csv a partir de los datos contenidos en la carpeta poscorp, generando el archivo dataset.csv .

El archivo datasetPP.csv contiene la data una ves eliminados los atributos que se consideraron contraproducentes en el analisis de datos.

El archivo datasetNor.csv contiene la data una ves que se le aplica una normalizacion.
